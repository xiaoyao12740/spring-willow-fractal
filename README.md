# Spring Breeze Willow Fractal Drawing

基于 MATLAB 的“春风杨柳”分形图形绘制与长度计算小项目。

## 项目简介

本项目使用 MATLAB 实现了一个具有树形递归结构的分形绘图程序。程序通过设定分枝层数、左分枝旋转角度和右分枝旋转角度，逐层生成类似“春风杨柳”形态的树状图案，并绘制出对应的分形图形。

除了图形绘制外，项目还分别采用：

- 直接求和方法
- 循环累加方法

对该分形结构的总长度进行计算与比较。

该项目适合作为 MATLAB 图形绘制、递归/迭代结构模拟与数列求和练习的小型课程项目。

## 功能说明

本项目主要实现了以下功能：

- 输入分枝层数
- 输入左、右分枝的旋转角度
- 自动绘制多层分形树状图
- 使用两种方法计算图形总长度
- 输出长度计算结果，便于比较不同层数下的变化规律

## 项目原理

项目将整棵“杨柳”图形视为由一条主干和多层分枝构成的递归结构。

### 1. 图形生成

初始主干由一条长度为 1 的线段表示。  
从第 2 层开始，每一层分枝由上一层线段经过以下变换得到：

- **左分枝**：按一定角度旋转，并按比例缩放
- **右分枝**：按另一角度旋转，并按比例缩放

在当前代码中：

- 左分枝缩放比例为 `0.9`
- 右分枝缩放比例为 `0.6`

程序通过旋转矩阵对线段进行几何变换，并不断迭代生成下一层分枝，最终绘制出整体分形图案。

### 2. 长度计算

项目还对该分枝结构的总长度进行了计算。  
根据代码中的求和形式，总长度可以表示为一个等比级数：

```text
L = 1 + 1.5 + 1.5^2 + ... + 1.5^(N-1)
````

其中 `N` 为分枝层数。

程序分别采用两种方式计算：

* **直接法**：使用符号求和 `symsum`
* **循环法**：通过 `for` 循环逐项累加

从而验证结果的一致性。

## 文件结构

* `stable/main.m`：稳定版本主程序，负责输入参数并调用绘图与计算函数
* `stable/plotcfyl.m`：绘制“春风杨柳”分形图案
* `stable/zhijie.m`：使用直接求和方法计算总长度
* `stable/cycle.m`：使用循环累加方法计算总长度

此外，仓库中还包含：

* `experimental/`：实验版或调试版代码
* `archive/`：旧版本脚本备份

如果只是正常运行项目，建议优先使用 `stable/` 目录中的代码。

## 运行环境

* MATLAB
* 若使用符号求和功能，需要支持 Symbolic Math Toolbox

## 使用方法

1. 打开 `stable` 文件夹；

2. 在 MATLAB 中将当前路径切换到 `stable` 目录；

3. 运行主程序：

   `main`

4. 按提示依次输入：

   * 分枝层数 `N`
   * 左分枝逆时针旋转角度
   * 右分枝逆时针旋转角度

程序会自动：

* 绘制对应的分形图形
* 输出直接法计算的总长度
* 输出循环法计算的总长度

## 输出内容

运行后通常可以得到以下结果：

* 一幅树形/柳枝式分形图
* 直接求和得到的总长度 `L`
* 循环计算得到的总长度 `L`

通过改变层数和旋转角度，可以观察：

* 图形结构如何变化
* 分形形态如何受参数影响
* 总长度如何随着层数增加而增长

## 项目特点

* 将图形绘制与数学求和结合在一起
* 使用矩阵旋转实现几何变换
* 结构清晰，适合用于 MATLAB 绘图练习
* 同时包含图形生成与数值/符号计算两部分内容
* 适合作为分形图、递归结构和可视化入门案例

## 项目局限性

* 当前项目主要是脚本式实现，没有图形界面
* 分枝比例在代码中写死，扩展性一般
* 主要用于演示分形绘制和求和思想，不属于复杂图形系统
* 若层数过大，图形会变得非常密集，显示效果可能受限
* 直接法依赖符号工具箱，部分环境下可能需要调整
